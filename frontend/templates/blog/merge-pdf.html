How to Merge PDF Files: Step-by-Step Guide

April 5, 20245 min read

Merging PDF files is one of the most common document tasks in any professional environment. Whether you're assembling a client proposal from separate sections, combining invoices for an accountant, or compiling a portfolio from individual project files, combining multiple PDFs into a single document makes sharing and filing much simpler. This guide shows you exactly how to do it and shares tips for getting the cleanest result.

Why Merge PDF Files?

Here are the most common reasons people need to combine PDFs:

  • Business proposals and reports — Cover pages, body sections, appendices, and CVs often exist as separate files that must be submitted as one document.
  • Legal and contract documents — A signed contract, supporting annexes, and identification documents may each be separate PDFs that need to be bundled together.
  • Accounting and invoicing — Monthly expense reports combining individual receipts and invoices into a single PDF for finance teams.
  • Academic submissions — Research papers, supporting data sets, and ethics approvals submitted as one combined document.
  • Photography and design portfolios — Multiple project files combined into a single presentation-ready PDF.
  • Scanned document archives — Individual scanned pages combined into one multi-page document.

What Happens When You Merge PDFs?

When you merge PDF files, all the pages from each source document are concatenated in the order you specify. The merged file contains every page from every source document. No content is changed, compressed, or lost during a standard merge — it's simply a repackaging operation.

The resulting file size is approximately the sum of all input file sizes (slightly more due to the merged PDF structure). If you want to reduce the size of the merged file, you can run it through the Compress PDF tool afterwards.

Page Order Matters

The order in which you upload files determines the page order in the merged PDF. If you upload File A, File B, then File C, the merged document will contain all pages of File A first, then all pages of File B, then all pages of File C. Take care to upload files in the exact order you want them to appear.

How to Merge PDFs on PDFWorks

  1. Go to PDF Tools and select Merge PDFs from the sidebar.
  2. Upload your PDF files. You can select multiple files at once from your file browser, or drag them all onto the upload zone.
  3. The files will be merged in the order they were selected. Check the order before proceeding.
  4. Click Convert. The merge is very fast — usually under 5 seconds for most documents.
  5. Download your merged PDF.

Tips for Merging PDFs

  • Name your files with numbered prefixes before merging — e.g., "01_cover.pdf", "02_body.pdf", "03_appendix.pdf" — so the sort order when selecting files matches the intended document order.
  • Check page orientation consistency. If some source PDFs are portrait and others are landscape, the merged document will alternate orientations. This is usually fine, but confirm it looks correct after merging.
  • Compress after merging if the combined file is large and you need to email it. Use the Compress PDF tool on the merged output.
  • Password-protected PDFs cannot be merged until they are unlocked. Use the Unlock PDF tool first if needed.
  • Verify the merged file before sending or archiving. Open it and scroll through every page to confirm nothing was missed or duplicated.

Merging vs Splitting

The opposite of merging is splitting — extracting specific pages from a PDF to create a smaller document. PDFWorks also offers a Split PDF tool for this purpose. You might split a 100-page PDF report into its individual sections, then remerge them in a different order, for example.

File Size Considerations

Merging PDFs does not change their internal content, so file sizes add together. A merge of four 5 MB PDFs will produce approximately a 20 MB output. If the resulting file is too large for email, run it through the Compress PDF tool. Medium quality compression on a 20 MB image-heavy merged PDF can often bring it down to 3–6 MB.
