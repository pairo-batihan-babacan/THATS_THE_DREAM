What This Tool Does

The Unlock PDF tool removes password protection from a PDF file. To use it, you must already know the correct password — this tool does not crack or bypass passwords on files you don't own. It simply provides a convenient way to produce an unlocked copy of a PDF you have legitimate access to, so you don't need to enter the password every time you open it.

How to Unlock a PDF on PDFWorks

  1. Go to PDF Tools and select Unlock PDF in the sidebar.
  2. Upload your password-protected PDF file.
  3. Enter the password for the file when prompted.
  4. Click Convert. The tool verifies the password and produces an unlocked PDF.
  5. Download the password-free version.

Common Legitimate Use Cases

  • Archived documents — You password-protected a document years ago and want to remove the password from your own archive.
  • Workflow simplification — You receive password-protected PDFs from a specific supplier and have the password — you want unlocked versions for your document management system.
  • Editing your own work — You previously locked a PDF to prevent edits, but now you need to update it. Unlock it first, then re-lock after editing.
  • Printing restrictions — The PDF has print restrictions (an owner/permissions password) that prevent printing. If you own the document, unlocking it removes these restrictions.

Important: Ethical and Legal Use Only

This tool is provided for legitimate use by document owners. Using it to bypass password protection on documents you do not own or have permission to access may constitute unauthorised computer access under applicable law (including the Computer Misuse Act in the UK, the Computer Fraud and Abuse Act in the US, and equivalent legislation elsewhere). PDFWorks does not condone or support misuse of this tool.

Two Types of PDF Passwords

PDFs can have two different types of password protection:

  • User/Open password — Required to open and view the document at all. You must provide this password to use the Unlock PDF tool.
  • Owner/Permissions password — Controls editing, printing, and copying rights. May not be required to open the file in a viewer.

Both types can be removed with this tool when you provide the correct password.
