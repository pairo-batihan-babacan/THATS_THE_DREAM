How to Merge PDFs on PDFWorks

  1. Go to PDF Tools and select Merge PDFs in the sidebar.
  2. Upload your PDF files. Select multiple files at once or drag them onto the upload zone. Files are merged in the order you select them.
  3. Click Convert. Merging is very fast — typically 2–5 seconds.
  4. Download your merged PDF file.

What Merging Does

The merge operation concatenates all pages from each source PDF in the order you specify. No content is modified, compressed, or removed during the merge. All original text, images, formatting, hyperlinks, and bookmarks are preserved. The resulting file size is approximately equal to the sum of the input file sizes.

Common Use Cases

  • Business proposals — Combine cover page, proposal body, appendices, and team CVs into one submission-ready document.
  • Contracts and legal documents — Bundle a signed contract with supporting annexes and identification documents.
  • Invoice collections — Combine monthly receipts and invoices into one PDF for accounting.
  • Academic submissions — Merge research paper, supporting data, and ethics approvals.
  • Portfolios — Combine individual project PDFs into a single downloadable portfolio.

Page Order Tips

Files are merged in the order they are uploaded. To ensure the correct page order, name your files with numeric prefixes (01_cover.pdf, 02_body.pdf, 03_appendix.pdf) before uploading so they sort correctly when selecting multiple files.

After Merging: Reduce File Size

If the merged PDF is larger than you need (for example, too large to email), run it through the Compress PDF tool. Medium quality compression typically reduces the merged file by 50–75% without noticeable quality loss.
