What Does "Protect PDF" Actually Do?

When you protect a PDF, the file's content is encrypted using AES-256 — the same standard used by governments, banks, and cloud storage providers. Without the correct password, the file cannot be opened, viewed, or read. It is not merely hidden; it is mathematically scrambled.

Specifically, when anyone opens the protected PDF in any viewer — Adobe Acrobat, Preview, Chrome's built-in viewer, mobile apps — they will see a password prompt before any content is shown. Entering the wrong password (or no password) shows nothing.

The Two Types of PDF Password

PDFs support two distinct passwords with very different behaviours:

  • Open Password (User Password) — Required just to view the file. This is what this tool sets. It is enforced by cryptography. Without it, the file content is inaccessible.
  • Owner Password (Permissions Password) — Controls what an authorised viewer can do: print, copy text, fill in forms, modify the document. This is NOT cryptographically enforced — some PDF tools can bypass it even without the password. It is "soft" access control, not a lock.

What Is AES-256?

AES-256 (Advanced Encryption Standard, 256-bit key) is a symmetric encryption algorithm standardised by the U.S. NIST and classified for top-secret government use. It has no known practical attacks — a brute-force attack on AES-256 is computationally infeasible with any hardware that exists or is expected to exist. Your PDF is safe as long as the password itself is strong (avoid short or common passwords).

PDFs can also use older, weaker encryption (RC4-128 or RC4-40). This tool always uses AES-256, the strongest option in the PDF specification.

What Can Go Wrong — Risks to Know

🔑 Forgotten password = permanent data loss. AES-256 has no back door. If you lose the open password, the file cannot be decrypted — not by PDFWorks, not by any tool, not by law enforcement without a court order and significant computing resources. Always keep a secure copy of the password (a password manager like Bitwarden or 1Password is ideal).
📱 Compatibility. Some very old PDF readers (pre-2008) do not support AES-256 and will refuse to open the file. All modern viewers — Adobe Acrobat, Apple Preview, Microsoft Edge, Chrome, Foxit — fully support it.
🖨 Owner password limitations. The Owner/Permissions Password prevents compliant viewers from printing or copying — but non-compliant tools and some PDF libraries can ignore permissions restrictions. Do not rely on permissions alone to prevent determined misuse.
