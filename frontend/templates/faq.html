Privacy & Files

PDF Tools

  • PDF compression reduces the quality and resolution of embedded images, removes redundant data, strips metadata, and optimises the internal structure. PDFWorks offers three quality levels: Low achieves maximum compression but may reduce image sharpness; Medium (recommended) balances file size with visual quality; High preserves maximum quality with modest reduction. PDFs that are mostly text compress less dramatically than image-heavy files.
  • A user password (open password) is cryptographically enforced — the file cannot be opened or read without it. PDFWorks uses AES-256 encryption for user passwords.

    An owner password controls permissions such as printing, copying, and editing. It is not cryptographically enforced and can be bypassed by most PDF readers and free tools. Do not rely on an owner password to prevent access to the content — use a user password for that.
  • No. If you have lost the user (open) password, the content is not recoverable through PDFWorks. AES-256 encryption is strong enough that brute-force recovery of a good password is not feasible. If the file had only an owner (permissions) password and no open password, the Unlock PDF tool can remove it.
  • No — and this is a common, dangerous misconception.
    Flattening converts interactive elements (form fields, annotations, drawn shapes) into static page content. It does not delete any underlying text or images. If you have drawn black boxes over text to "redact" it, the text is still in the file after flattening. Anyone can recover it by removing the annotation layer or extracting the text stream with free tools.

    True redaction requires a tool that permanently deletes the underlying content (such as Adobe Acrobat Pro's Redact feature), followed by flattening, followed by verifying the text is no longer selectable.

Complex documents with columns, tables, or unusual fonts may lose some formatting. Use original Word or editable PDF source files where possible, and expect some manual cleanup after conversion.

Audio Tools

  • PDFWorks supports MP3, WAV, OGG, FLAC, AAC, M4A, and AIFF for both input and output. You can convert between any of these, compress, trim, normalize loudness, change playback speed, or extract audio from a video file.
  • No. The Change Speed tool uses time-stretching (ffmpeg's atempo filter), which adjusts playback duration without altering pitch. A podcast sped up to 1.5× will play faster but will not sound higher-pitched.
  • WAV is uncompressed PCM audio — there is no such thing as a compressed WAV that stays in WAV format. When you compress a WAV file on PDFWorks, it is transcoded to MP3 at the chosen bitrate, producing a much smaller file. If you need lossless compression, convert to FLAC instead.
  • Normalization adjusts the overall loudness of an audio file to a target level measured in LUFS (Loudness Units Full Scale). Use it when:
    • You have recordings at different volumes and want them consistent
    • You are uploading to a streaming platform and need to hit their loudness target (−14 LUFS for Spotify/YouTube, −16 LUFS for general use, −23 LUFS for broadcast)
    PDFWorks uses the EBU R128 loudnorm filter, the industry standard for broadcast and streaming loudness.
  • Yes. The Extract Audio tool accepts MP4, MOV, MKV, AVI, and WebM files and outputs the audio track in your choice of format: MP3, WAV, OGG, FLAC, AAC, or M4A.

Video Tools

  • Video input accepts MP4, MOV, MKV, AVI, and WebM. All video output is produced as H.264/AAC MP4, which plays natively on all major browsers, operating systems, and devices without plugins.
  • MP4 with H.264 video and AAC audio is the universal web-safe format. MKV, AVI, and other containers — or codecs like H.265 and VP9 — cause playback failures on many devices and browsers. Standardising to H.264 MP4 guarantees your output plays everywhere without extra software or conversion on the recipient's side.
  • Results vary by source material, but typical ranges:
    • High quality — ~10–20% smaller, visually identical
    • Medium quality — ~40–60% smaller, minor quality trade-off
    • Low quality — ~70–80% smaller, visible quality reduction
    Screen recordings and animation compress far better than fast-moving footage such as sport or film.

General

  • No account is required. You can use every tool on PDFWorks without signing up, providing an email address, or logging in. Simply upload your file, process it, and download the result.
  • The Pro token is a one-time payment of $5 that unlocks higher file size limits (up to 50 MB per file) and removes ads for 30 days. Unlike a subscription, it is a single purchase — there is no recurring billing and no account required. The token is linked to your browser session.
  • Yes. All tools are fully functional on modern mobile browsers (Safari on iOS, Chrome on Android). File upload, processing, and download all work on mobile. Some interactions — such as the rotate PDF page grid with multi-select — work best on desktop with a mouse and keyboard.

  • Processing time depends on file size and tool complexity. PDF compression of a 1 MB document completes in seconds. Audio normalization, video re-encoding, and OCR on large files may take up to a minute. The status indicator on each tool page updates automatically — you do not need to refresh.
