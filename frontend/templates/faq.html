Frequently Asked Questions

Everything you need to know about using PDFWorks. Can't find your answer? Contact us.

  • Yes, PDFWorks is completely free to use. All tools — PDF compression, conversion, merging, image editing, audio conversion, and more — are available at no cost. We cover our operating expenses through non-personalised display advertising.
  • No account is required. You can use every tool on PDFWorks without signing up, providing an email address, or logging in. Simply upload your file, process it, and download the result. There is nothing to install.
  • Yes. All file uploads and downloads are protected with HTTPS encryption in transit. Your files are processed on secure servers and never read, shared, or sold. Everything is automatically and permanently deleted within 30 minutes of upload. We have no interest in the content of your files.
  • All uploaded files and conversion outputs are automatically and permanently deleted within 30 minutes of upload. This applies to both the original file and the processed result. This deletion policy cannot be extended or paused — it is a core privacy feature, not a configurable option.

  • Free users can upload files up to 20 MB. If you need to process larger files (up to 50 MB), you can purchase a one-time Pro token for $5 which is valid for 30 days. There is no account required even for the Pro option.
  • PDF compression works by reducing the quality and resolution of embedded images, removing redundant data, stripping metadata, and optimising the internal PDF structure. PDFWorks offers three quality levels: Low achieves maximum compression but may reduce image sharpness; Medium (recommended) balances file size with visual quality; High preserves maximum quality with modest file reduction. The actual size reduction depends on how many images your PDF contains — PDFs that are mostly text compress less dramatically than image-heavy files.
  • It depends. If your PDF was created digitally (from Word, a web browser, etc.) it contains actual text and can be converted to an editable Word document quite accurately. If your PDF is a scanned image of a physical document, the text is embedded as an image and requires OCR (Optical Character Recognition) to extract. Our Document → OCR tool can help extract text from scanned images. For best results with scanned PDFs, ensure the scan is high resolution (300 DPI or above) and the text is clearly legible.
  • Go to PDF Tools and select "Merge PDFs". You can upload multiple PDF files at once and the tool will combine them into a single PDF in the order you upload them. The merged file retains all original pages from each source document.
  • From PDFs: Author name, creation date, modification date, application name (e.g. "Microsoft Word 2019"), operating system info, and custom properties.
    From images: EXIF data including GPS coordinates (location), camera make and model, lens information, date and time taken, photographer name, copyright strings, and thumbnail previews.
    From audio files: ID3 tags including artist, album, title, year, genre, and embedded artwork.
  • Yes, PDFWorks is fully responsive and works on smartphones and tablets. All processing happens on our servers, so even large or complex conversions work regardless of your device's processing power. For the best experience on mobile, use a modern browser such as Chrome, Safari, Firefox, or Edge.
  • PDF is a fixed-layout format that does not store information about text flow, styles, or layout intent — it just stores the final visual result. Converting a PDF back to an editable format is technically complex and involves reverse-engineering the original layout. Complex documents with columns, tables, headers, or unusual fonts may lose some formatting. For best results: use original Word or editable PDF source files where possible, and expect some manual cleanup after conversion. Scanned PDFs will need OCR first.
  • Go to PDF Tools and select "Unlock PDF". You can use this tool to remove the password from a PDF file that you own and have the original password for. Please note: this tool is intended for legitimate use cases such as unlocking your own archived files. Attempting to bypass passwords on files you do not own may violate copyright or computer misuse laws.
  • An API is not publicly available at this time. If you are interested in API access for business use — for example, to integrate PDFWorks conversion tools into your own application — please contact us to discuss options.
  • The Pro token is a one-time payment of $5 that unlocks higher file size limits (up to 50 MB per file) and removes ads for 30 days. Unlike a subscription, it is a single purchase — there is no recurring billing and no account required. The token is linked to your browser session.
  • The Merge PDFs tool accepts multiple files simultaneously. Most other tools process one file at a time. Batch conversion for multiple individual files is on our roadmap. In the meantime, conversions are fast — typically a few seconds per file — so processing files one by one is usually practical for small batches.
